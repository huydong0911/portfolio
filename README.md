# Portfolio Cá Nhân
Website giới thiệu kỹ năng Frontend, các dự án thực tế và học vấn của cá nhân.

## 🚀 Công nghệ sử dụng
* Lõi ứng dụng: **ReactJS**
* Build tool: **Vite**
* Triển khai: **GitHub Pages**

## 🔗 Liên kết trực tiếp
[Xem trực tiếp Website Portfolio tại đây](https://huydong0911.github.io/portfolio)

## 📁 Cấu trúc thông tin
1. Giới thiệu bản thân và mục tiêu nghề nghiệp.
2. Dự án tiêu biểu (HomeMarket, Techmart, MOMOCHA).
3. Chứng chỉ chuyên môn.
4. Thông tin liên hệ.
